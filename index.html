The Product

TaskRiver is productivity-focused desktop application, allowing users to engage in task completion through an intuitive flow-based task chaining process.

Create

Create Tasks that you need to complete.

View

View other tasks that need to be completed other than the one in progress. You can even click on them to see everything that needs to be done.

Flow

Chain smaller subtasks to a bigger task and set your own dependencies.
